## 实现要点

### 翻译缓存（F1）
- 键：`md5(image_path:mtime)`，上限 500 条，LRU 淘汰
- 存储：项目根目录 `translation_cache.json`
- API 异常时自动降级查询历史缓存（`get_fallback`）
- 错误缓存过滤（429 限流文本等）

### 压缩包支持（F2）
- `.cbz` / `.zip`：Python 标准库 `zipfile`，仅解压图片，排除 __MACOSX/
- `.cbr`：内嵌 `bin/7z.exe` + `bin/7z.dll`（LGPL，~1.1MB），`subprocess` 调用 `7z e` 扁平解压
- 统一入口 `_load_source(path, is_archive)`，目录/压缩包共用后续处理管道

### 阅读位置记忆（F7）
- 键：`_source_path`（对压缩包稳定，不受临时目录路径变化影响）
- 上限 20 条，dict 插入顺序实现 LRU 淘汰
- 保存时机：切换文件夹时 + 关闭窗口时
- 恢复时机：`load_file_list()` 末尾，越界/不存在时回退第一页

## 验证收口

- F1：缓存命中跳过 API、清除缓存后重新触发 API、错误缓存自动过滤
- F2：.cbz/.zip/.cbr 均可打开、翻页翻译正常、临时目录自动清理、最近菜单/自动恢复正确识别 CBR
- F3：状态栏分阶段提示、按钮省略号动画、骨架屏脉冲动画
- F4：首次引导浮层、状态栏常驻快捷键提示
- F5：搜索框实时过滤、Ctrl+F/Esc 快捷键
- F6：下拉菜单显示最近 10 条、点击打开、清除历史
- F7：关开程序自动恢复页码、切换文件夹保存旧位置

## 本阶段收益

- 用户可以多次重启程序而无需重新翻译（缓存持久化）
- 支持漫画行业主流分发格式（.cbz/.zip/.cbr），适配 90%+ 的漫画资源
- 翻译等待过程有可见反馈（骨架屏 + 分阶段提示）
- 阅读路径完整闭环：打开 → 自动跳转上次位置 → 翻页 → 切换 → 再次打开
