## 目标

支持 `.cbz` / `.zip` 格式漫画压缩包，解压到临时目录阅读，关闭时自动清理。

## 技术决策

- **格式范围**：仅 `.cbz` + `.zip`，使用标准库 `zipfile` 零依赖；`.cbr` 暂不支持
- **解压策略**：`tempfile.mkdtemp(prefix="bubbletrans_arc_")` 每次新建临时目录；切换来源/关闭程序时 `shutil.rmtree` 清理
- **过滤规则**：排除 `__MACOSX/`、`.DS_Store`、非图片文件
- **缓存兼容**：暂不处理（每次解压路径不同，缓存无法跨次命中，压缩包场景不会频繁开关）
- **安全防护**：`is_zipfile()` 兜底校验，解压异常 → `QMessageBox.critical` 提示

## 涉及文件

| 文件 | 改动类型 |
|------|----------|

## 任务拆解

### 1.1 创建 `src/utils/archive.py`

**常量**：
- `ARCHIVE_EXTS = ('.cbz', '.zip')`
- `IMAGE_EXTS = ('.jpg', '.jpeg', '.png', '.webp', '.bmp')`
- `TEMP_DIR_PREFIX = "bubbletrans_arc_"`

**函数**：
- `is_archive(path) -> bool`：后缀 + `zipfile.is_zipfile()` 双重校验
- `extract_to_temp(archive_path) -> tuple[str, list[str]]`：
  1. 校验文件格式
  2. `namelist()` 获取文件列表
  3. 过滤保留图片文件，排除垃圾
  4. 排序
  5. `mkdtemp` 创建临时目录
  6. 逐过滤文件 `extract()`
  7. 返回 `(temp_dir, sorted_image_names)`

### 1.2 修改 `src/ui/window.py`

**新增 import**：
- `import shutil`
- `import zipfile`
- `import utils.archive as archive`

**`__init__` 新增状态**：
- `self._archive_temp_dir = ""`（非空表示当前来源是压缩包）

**`init_ui` 新增动作**：
- "Open Archive" action → 连接 `open_archive` 槽
- 文件对话框过滤 `Comic Archive (*.cbz *.zip)`

**新增方法**：
- `open_archive()`：弹出文件选择 → 调用 `_load_source(path, is_archive=True)`
- `_load_source(path, is_archive)`：统一入口
  - 清理上次临时目录（`_cleanup_archive_temp()`）
  - 压缩包分支：调用 `archive.extract_to_temp()`，设置 `self._archive_temp_dir`
  - 文件夹分支：直接设置 `self.current_folder`
  - 统一后续：`_clear_image_cache()` → `load_file_list()` → 状态栏更新
- `_cleanup_archive_temp()`：`shutil.rmtree(self._archive_temp_dir, ignore_errors=True)`

**修改现有方法**：
- `open_folder()`：选择后调用 `_load_source(folder, is_archive=False)` 替代内联逻辑
- `closeEvent()`：增加 `_cleanup_archive_temp()` 调用

**状态栏文案**：
- 文件夹来源：`已加载: {name} ({n} 张)`
- 压缩包来源：`📦 {name} ({n} 页)`

### 1.3 改动范围外（不动的部分）

- `load_file_list()`：利用 `os.listdir` 对临时目录透明，无需修改
- 翻页、预加载、翻译回调：image_path 统一指向临时目录内文件，逻辑不变
- 图片 LRU 缓存：key 为临时目录路径，单次会话内有效
