## 目标

向用户展示可用的快捷键，降低上手门槛。通过首次引导浮层 + 状态栏常驻提示 + Settings tooltip 三层覆盖。

## 涉及文件

| 文件 | 改动类型 |
|------|----------|
| `src/ui/window.py` | 快捷键引导浮层、状态栏提示、Settings tooltip |
| `src/ui/canvas.py` | 确认现有快捷键定义位置 |

## 现有快捷键

需要先确认 `ImageCanvas` 中已实现的快捷键（从 PDR 和代码推断）：
- `←` / `→`：翻页
- 右键拖拽：框选翻译区域
- `Ctrl + 滚轮` / `滚轮`：缩放

## 任务拆解

### 4.1 首次启动快捷键引导浮层
- 在 `MainWindow` 中添加 `ShortcutOverlay`（`QDialog` 或 `QWidget` + `QGraphicsOpacityEffect`）
- 内容：列出所有快捷键及功能说明，带"不再提示"复选框
- 触发条件：`config.json` 中 `shortcut_hint_shown` 字段不存在或为 `false`
- 关闭后写入 `shortcut_hint_shown: true`
- 样式：半透明遮罩 + 居中卡片，简洁明了

快捷键列表：
| 快捷键 | 功能 |
|--------|------|
| `←` / `→` | 上一页 / 下一页 |
| 右键拖拽 | 框选翻译区域 |
| 滚轮 | 缩放图片 |
| `Esc` | 取消框选 |
| `F5` | 翻译当前页（若有） |

### 4.2 状态栏常驻快捷键提示

- 状态栏右侧（或固定位置）添加 label：`← → 翻页 | 右键框选 | 滚轮缩放`
- 使用 `QStatusBar.addPermanentWidget()` 保证位置固定
- 字体略小于状态栏主文本，颜色偏灰

### 4.3 Settings 按钮 tooltip

- Settings 按钮添加 `setToolTip("设置 (快捷键、API 配置)")`
- hover 时显示简要说明

## 验收标准

- [ ] 首次启动（或 `shortcut_hint_shown` 为 false）时弹出快捷键引导浮层
- [ ] 勾选"不再提示"并关闭后，下次启动不再弹出
- [ ] 状态栏始终显示核心快捷键提示（`← → | 右键框选 | 滚轮缩放`）
- [ ] Settings 按钮 hover 时显示 tooltip
- [ ] 引导浮层样式美观、与主窗口风格一致
- [ ] 快捷键描述与实际功能一致
